Hexad(ヘキサ)の SDK のダウンロードについて説明します。

ベータ版では、[こちらのフォーム](https://form.run/@admin-1595728115)入力後にメールにて SDK をダウンロードできるリンクを送らせていただきます。

## サポートしている Unity バージョン

Unity のバージョンと SDK のサポートしているバージョンが一致していないと、不具合が発生する可能性があります。

- Unity 2018.4.X
- Unity 2019.X.Y
