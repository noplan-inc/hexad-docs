Unity に SDK をインストールする方法について説明します。

広告を配置するシーンを Unity で開いてください。ダウンロードした SDK をダブルクリックするとインポートするファイルを選択する画面が出てくるので、全てをインポートします。

これには数分かかることがあります。

<img
alt="SDK をインポートする"
src={useBaseUrl("img/getting-started/import-sdk.png")}
/>

これでインストールは完了です。
