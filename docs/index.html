Hexad（ヘキサ）とは#

Hexad は、ユーザにとって邪魔にならない広告で、ゲーム開発者がスマートにコンテンツを収益化できるようにするための独自のプラットフォームです。

始めましょう#

このチュートリアルを見て、HexadSDK を Unity に導入し、広告を設置し、マネタイズを 2 分以下で始めましょう。

/ TODO チュートリアル動画 /

  1. 開発者として Hexad に登録して、HexadSDK を入手する。
  2. シーンに広告を設置する。
  3. ３営業日以内に審査を行われ、収益が発生し始めます。
